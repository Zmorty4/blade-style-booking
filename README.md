# The Gentlemen's Lounge inspired website

A responsive static barber shop website inspired by https://www.thegentslounge.co.uk/.

## Files

- `index.html` - page structure and content
- `styles.css` - responsive dark vintage styling
- `script.js` - mobile navigation and booking form feedback

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.

## Notes

The site uses original HTML/CSS/JS and references public imagery from the original Webflow site for visual similarity. Replace phone, address, and booking form handling with production details before launch.
